# Souradeep De — Graphics Portfolio

A clean, conversion-focused portfolio website for **Souradeep De**, a Canva Designer & Visual Content Creator.

## Features
- 🎨 Modern, responsive single-page design inspired by Cozydiadora
- 🖼️ Filterable portfolio grid showcasing design projects
- 📱 Fully mobile-responsive with hamburger navigation
- ✉️ Contact form with client-side validation
- 🎯 Scroll reveal animations and micro-interactions
- ♿ WCAG-compliant accessibility

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Inter)
- No framework dependencies

## Live Preview
Open `index.html` in a browser to view the portfolio.
